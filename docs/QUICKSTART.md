# VibeOps Quickstart

This guide will take you from installation to your first successfully promoted ticket in under 10 minutes.

## Prerequisites
- Node.js >= 18
- An autonomous coding agent installed on your machine (e.g. Claude Code, Antigravity, or Codex)

## 1. Start VibeOps
Run the VibeOps web relay from the terminal. This launches the application on `http://localhost:3000`:
```bash
npm start
```

## 2. Onboarding Wizard
When you open VibeOps for the first time, you will be greeted by the Onboarding Wizard.

### Step 1: Create your project
- Enter your project's name.
- (Optional) Browse and select the absolute path to your local repository.
- Click **Create Project**.

### Step 2: Agent Detection
- VibeOps will automatically scan your system for installed autonomous agent CLIs.
- Once it detects an agent, click **Generate relay.json & Continue**. This writes your configuration file so VibeOps knows how to route tickets to your local agents.

### Step 3: First Ticket
- Write a short **Title** and **Description** for your first task.
  - *Example Title:* "Add a health check endpoint"
  - *Example Description:* "Add a simple GET /health endpoint that returns { status: 'ok' }"
- Click **Start Building**.

## 3. The Forge (Execution)
You will be deep-linked directly to the Forge with your new ticket selected.

- Click **Run Pipeline** in the top right to start the autonomous agent.
- VibeOps provisions a sandbox and delegates the task to the agent.
- Watch the output stream in real-time as the agent researches, plans, and writes code to fulfill your request.

## 4. Review & Promote
- Once the agent finishes, the ticket status changes to `review`.
- Review the diff generated by the agent in the right-hand panel.
- If everything looks good, click **Promote** to merge the changes into your main branch.

🎉 **Congratulations!** You've just automated your first coding task with VibeOps.
